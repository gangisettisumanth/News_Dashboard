# News Dashboard App

## Overview

The News Dashboard App is a React application that fetches and displays top news articles from around the world. It utilizes the [News API](https://newsapi.org/) to provide real-time news data.

## Features

- **Country Selection**: Users can choose a specific country to view top news articles from that region.
- **Pagination**: The app supports pagination, allowing users to navigate through multiple pages of news articles.
- **Responsive Design**: The user interface is designed to be responsive and accessible on various devices.
